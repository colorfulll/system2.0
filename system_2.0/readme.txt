可直接运行的文件在debug文件夹下，即trail.exe。

两个源代码文件、两个头文件分别为main.cpp、help.cpp、main.h、help.h

该程序目前支持：
普通用户功能
1.新用户注册（只有注册后才能使用管理物品的功能）
2.用户登录（注册被同意过后）
登录后可以使用
1.添加物品（共有三种物品，普通物品，书籍和食物）。
2.删除物品；输入物品种类、物品名称，拥有者和数量。
3.查找物品；通过物品名称。
4.显示物品列表。
5.可以保存数据。

管理员功能：
1.登录
2.登录后会自动弹出新用户注册申请，等待管理员同意

测试方案：
1.测试注册
2.先创建一个管理员的账户，测试管理员的同意功能
3.同意后，测试普通用户的登录
4.普通用户登录后，先看一下物品列表
5.添加一个物品（6个），然后再显示物品列表
6.再删除1.删除一个没有添加过的物品，2.这个物品（3个），然后再显示物品列表
